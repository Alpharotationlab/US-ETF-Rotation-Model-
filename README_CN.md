## 概述

本倉庫基於以下核心邏輯提供美股ETF輪動框架的簡化研究版：

- 相對動量排序（20日回看）
- 200日移動平均趨勢過濾
- 每5個交易日調倉
- 0.10% 單邊交易成本假設

項目包含：

- 歷史回測引擎
- 基於 `yfinance` 的數據獲取
- 績效評估指標
- 多隻流動性良好的美股ETF
- 簡化研究實現

本項目的目標是提供一個透明、可復現、可驗證、值得信任的系統化美股ETF配置研究框架。

---

## 研究理念

本倉庫並非試圖提供所謂「神奇策略」，而是希望建立一個清晰、透明、可復現、可驗證的ETF輪動研究框架。

簡化研究版重點強調：

- **清晰性** — 代碼優先考慮人類可讀性
- **可復現性** — 執行 `python run_backtest.py` 可獲得一致結果
- **教育價值** — 幫助理解動量排序與趨勢過濾如何協同運作

擴展研究版則在此基礎上增加：

- 多因子排序（不限於動量）
- 市場狀態識別
- 動態風險管理
- 自適應倉位控制
- 每週模型更新與研究跟蹤

開源版本本身是一個真實、完整且可運行的研究框架，而不是單純展示用途的「閹割版」。

## 為什麼選擇ETF輪動？

相比個股策略，ETF輪動具有以下優勢：

- 分散單一公司風險
- 避免財報暴雷
- 更適合系統化交易
- 回測穩定性通常高於個股模型
- 更容易控制流動性與交易成本

美股市場長期存在行業風格輪動、利率週期切換、成長／價值風格轉換以及風險偏好變化，不同ETF之間的長期表現差異可能非常明顯。

ETF輪動策略的核心目的不是預測市場，而是透過規則化方法，系統性跟隨相對強勢資產。

## 美股ETF市場特點

相比A股市場，美股ETF市場具有以下特點：

| 特點 | 美股ETF市場 |
|---|---|
| 機構參與度 | 高 |
| 長期趨勢 | 較穩定 |
| ETF流動性 | 極高 |
| 行業覆蓋 | 完整 |
| 全球資金影響 | 顯著 |
| 長期牛市特徵 | 明顯 |

因此，美股ETF輪動策略更偏向趨勢跟隨、資產輪動與風險調整收益，而不是短期題材交易。

## 策略邏輯

簡化模型執行以下步驟：

1. 從多隻流動性良好的美股ETF中構建候選池
2. 根據過去20個交易日收益率進行動量排序
3. 當前價格必須高於200日均線
4. 持有滿足條件且動量最強的前3隻ETF
5. 每5個交易日調倉一次

本開源版使用動量輪動研究中合理且廣泛接受的默認參數，鼓勵使用者自行研究與測試不同參數設定。

## 常見問題（FAQ）

### 1）為什麼有些年份會跑輸SPY？

輪動策略並不會在所有年份持續超越大盤。當美股權重股全面上漲、AI／科技單邊行情極強、市場呈現高度集中牛市時，分散式輪動模型可能落後於SPY或QQQ。

本模型更關注長期風險收益比、回撤控制與策略穩定性，而不只是短期收益排名。

### 2）為什麼使用200日均線？

200日均線是全球趨勢跟蹤策略中最經典的長期趨勢過濾工具之一。其目的不是預測市場頂部或底部，而是盡量避免長期空頭趨勢，改善組合的整體風險收益比。

### 3）為什麼只持有3隻ETF？

較集中的持倉可以提高動量暴露與輪動效率，但同時也會帶來更高的波動率與回撤風險。完整版模型會透過市場狀態識別與風險預算進一步改善這一問題。

## 風險提醒

ETF輪動策略並非低風險策略。本模型可能面臨趨勢失效、行業快速反轉、高波動市場、流動性衝擊、宏觀政策風險與長時間回撤等情況。

尤其在加息週期、流動性收縮、高估值修正等市場環境下，動量策略可能出現明顯回撤。

任何歷史回測均不代表未來實際收益。

## 使用方法

```python
from model import USETFRotationModel

model = USETFRotationModel(
    universe=[
        'SPY', 'QQQ', 'IWM',
        'XLK', 'XLE', 'XLF',
        'GLD', 'TLT'
    ],
    lookback_days=20,
    ma_period=200,
    top_n=3,
    rebalance_freq=5
)

results = model.backtest(
    start='2019-01-01',
    end='2026-04-30'
)

results.plot_equity_curve()
results.print_summary()
```
